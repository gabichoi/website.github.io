Projects

These are some of my favorite projects I worked on.

Corody

Description:

As orders of shelter-in-place amid COVID-19 outbreak were issued, two of my friends and I developed a donation website to help the homeless community in Berkeley. The platform consists of getting supplies and funding from users in order to build and share hygiene kits.

Technical Features:

HTML | CSS | PHP | Illustrator

Cheep Cheep Stay Dry!

Description:

First game I built from scratch! A collision dectecting square turned into a little chick avoiding collisions from raindrops. The level of difficulty increases in proportion with the user's score.Try to keep her dry!

Technical Features:

Python | Pygame | Illustrator

Pacman AI

Description:

A truly challenging project in which I learned AI strategies to make my Pacman agent "smarter" in its complex maze world. After 80 runs of trial-and-error, Pacman was fully trained to outwit the ghosts, as refelected in this recording.

Technical Features:

Python | Reinforcement Learning | Q-learning

BearMaps

Description:

In this project I built the back-end of a web application that supports an interactive map of Berkeley. The application can handle features such as autocompleting query responses, zooming in and out, and finding the shortest path between two locations.

Technical Features:

Java | Open Street Maps API | Trie DFS | A* Algorithm
